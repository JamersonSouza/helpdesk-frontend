Olá! Seja bem vindo(A) ao sistema Help Desk desenvolvido por mim.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular e Spring Boot.
O sistema está rodando na nuvem sendo o front futuramente na Vercel e o back na Heroku.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.

Fique a vontade para testar o sistema e deixar sua sugestão.
